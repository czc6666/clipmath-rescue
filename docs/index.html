适用于带 data-formula 的微信公众号数学文章。点击一次，把 SVG 公式改成 Markdown LaTeX，再继续用你原来的 Obsidian Web Clipper。

1. 把按钮拖到书签栏

抢救本页公式

手机或无法拖拽？右键复制链接地址，新建书签后粘贴到网址栏。

2. 在含公式文章中使用

  1. 打开要保存的微信公众号文章。
  2. 点击书签栏里的“抢救本页公式”。
  3. 看到“已转换 N 个公式”后，再点击原来的网页剪藏器。
